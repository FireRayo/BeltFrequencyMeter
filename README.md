## ðŸ”§ How to use

**Get the tension value:**  
1. Place the microphone very close to the band and tap it to make it vibrate while recording audio.  
2. Open the audio in the app and press ***Analyze***.
3. You can also use it to tune musical instruments.
4. Select whether you want to see the value in ***Hz*** or in ***Note***.
